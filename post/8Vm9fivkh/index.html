一、图的概念

图是一个二元组，由 点集 (Node set) 和 边集 (Edge set) 组成，即 G=(V(G),E(G))G = (V(G), E(G))G=(V(G),E(G))。
常用 G=(V,E)G = (V, E)G=(V,E) 表示图。

二、图的分类

自环 (Loop)：对 EEE 中的边 e=(u,v)e = (u, v)e=(u,v)，若 u=vu = vu=v，则 eee 被称作一个自环。

重边 (Multiple edge)：若 EEE 中存在两个完全相同的元素（边）e1,e2e_{1}, e_{2}e1​,e2​，则它们被称作（一组）重边。

若一个图中有自环或重边，它被称为 多重图 (Multigraph)，否则被称为 简单图 (Simple graph)。

三、点的度数

与一个顶点 vvv 关联的边的条数称作该顶点的 度 (Degree)，记作 d(v)d(v)d(v)。特别地，对于边 (v,v)(v, v)(v,v)，即自环，则每条这样的边要对 d(v)d(v)d(v) 产生 222 的贡献。

在有向图 G=(V,E)G = (V, E)G=(V,E) 中，以一个顶点 vvv 为起点的边的条数称为该顶点的 出度 (Out-degree)，记作 d+(v)d^{+}(v)d+(v)。以一个顶点 vvv 为终点的边的条数称为该节点的 入度 (In-degree)，记作 d−(v)d^{-}(v)d−(v)。显然 d+(v)+d−(v)=d(v)d^{+}(v) + d^{-}(v) = d(v)d+(v)+d−(v)=d(v)。

0x01 图的存储

常用的图的存储方式有以下几种：

  1. 邻接矩阵
  2. 邻接表
  3. 链式前向星

一、邻接矩阵

存储方法

使用 g[][] 存储图，若 g[u][v] 为 truetruetrue 则代表有一条从 uuu 到 vvv 的边。若为带边权的图，则使用 g[u][v] 存储从 uuu 到 vvv 的边的边权。

若为无向图，则 g[][] 斜对称，即 g[u][v]=g[v][u]g[u][v] = g[v][u]g[u][v]=g[v][u]。

参考代码（以有向带边权图为例）

int g[N][N];
bool vis[N][N];

void add_edge(int u, int v, int w){
	g[u][v] = w;
	// g[v][u] = w;    若为无向图则加上这一句 
}

void dfs(int x) {
	vis[x] = true;
	for(int y = 1; y <= n; v++) {
		if(g[x][y] && !vis[x]){
			dfs(y);
		}
	}
}

复杂度

遍历图的时间复杂度：O(n2)O(n^{2})O(n2)
空间复杂度：O(n2)O(n^{2})O(n2)
